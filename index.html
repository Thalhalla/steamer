Steamer

This is a generic docker container that will enable you to run steam servers in the cloud

Requirements

Docker 1.4.1+ Everything else runs in the container

Usage

First download using the above links or clone the repo directly

Then replace steamer.txt with a custom script of your own

view the dev docs to create your own here that is the same as the default steamer.txt given here

Then type make run

now type docker ps you should see your container running
